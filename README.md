# IOT47_ESP_SIM7600
Thư viện LTE SIMCOM cho ESP32
Các giap tiếp hỗ trợ gồm:
- SMS
- CALL
- MQTT(s)
- Firebase
- HTTP(s)
- NTP
- FTP (file tranfer)
- OTA

Sử dụng: Xem ví dụ

Đây là thư viện giao tiếp với module dòng 76XX, nó được thiết kế chủ yếu theo cơ chế bất đồng bộ với các cuộc gọi lại (Callback)
